Frequently Asked Questions

What if I lose my card?
Each LockStone can store up to 50 authorized cards. You get multiple cards/pendants with each unit, and you can easily add/remove cards through the admin interface. Lost cards can be instantly deauthorized.
Does it work on Windows/Mac/Linux?
Yes! LockStone uses USB-HID protocol and works on Windows 10/11, macOS 10.14+, and most Linux distributions. No drivers needed - just plug and play.
Can I customize the LED patterns?
Absolutely. The Python SDK gives you full control over the 16 NeoPixel LEDs. Create custom animations, colors, and patterns for different users or events.
How secure is RFID compared to passwords?
LockStone uses AES-256 encrypted communication and secure authentication protocols. Physical cards are much harder to compromise than passwords, and lost cards can be instantly deauthorized.
Can I integrate it with existing systems?
Yes! The Python SDK allows integration with Active Directory, LDAP, time tracking systems, and custom applications. Full API documentation included.
